# Lynx

 > A flexible web-based code editor

## Features

- Live Edits, much like Google Drive
- Git Version Control Support
- File backups
- Many code themes
- Secure Logins
- Permissions System

## Installation

You can install Lynx with NPMjs

`npm-install lynxeditor`

You can also download the Lynx Editor [here]().
